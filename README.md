# Dofoto PRD 共享

PRD HTML 文件托管 + GitHub Pages 在线预览。

## 在线访问

- 入口：https://crystal310.github.io/dofoto-prd-share/
- 最新版本：https://crystal310.github.io/dofoto-prd-share/dofoto%20v1.320.0.html

## 更新流程（PM）

1. 本地修改 HTML 文件
2. `git add -A && git commit -m "update vX.Y.Z"`
3. `git push`
4. GitHub Pages 自动重新部署（约 30 秒~2 分钟）
5. 通知开发刷新页面
